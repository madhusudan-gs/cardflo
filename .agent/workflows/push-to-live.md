// turbo-all
1. Add all changes to git
```bash
git add .
```

2. Commit changes with a descriptive message
```bash
git commit -m "Update from Antigravity: [Current Date/Time]"
```

3. Push to GitHub
```bash
git push origin main
```

4. Monitor the deployment
Go to https://github.com/madhusudan-gs/cardflo/actions to see the progress.
